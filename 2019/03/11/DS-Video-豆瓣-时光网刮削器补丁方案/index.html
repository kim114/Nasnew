DS Video 豆瓣+时光网刮削器补丁方案
2019-03-11

常常使用Video收藏存储电影的朋友常常会出现一些尴尬的事情
有些资源的图片无法通过Video自带的搜刮器匹配图片
如果使用国内资源的话相对要好很多
开始方法
打此补丁，即可在搜索影片资料中，选 简体中文 即为豆瓣/时光网电影的内容。

安装方法：

1：开启DSM的ssh，并登入
2、执行一句话安装：
豆瓣源：

sudo wget -N --no-check-certificate https://sh.9hut.cn/dsvp.sh && sudo bash dsvp.sh install

时光网源：

sudo wget -N --no-check-certificate https://sh.9hut.cn/dsvpmt.sh && sudo bash dsvpmt.sh install

卸载方法：

1：开启DSM的ssh，并登入
2、执行一句话卸载：
豆瓣源：

sudo wget -N --no-check-certificate https://sh.9hut.cn/dsvp.sh && sudo bash dsvp.sh uninstall

时光网源：

sudo wget -N --no-check-certificate https://sh.9hut.cn/dsvpmt.sh && sudo bash dsvpmt.sh uninstall

一些说明：
1、此补丁仅适用于dsm6.x，目前包含6.0-6.2；
2、自动索引目录欲使用豆瓣/时光网补丁，请设置目标目录默认为“简体中文”；
3、豆瓣电影不区分电影还是连续剧，目前此补丁只针对电影类使用（我目前是电影和连续剧放一起）；
4、手动索引会产生.vsmeta信息文件，此为ds默认行为，与补丁无关；
