使用群晖Nas的git拉取远程仓库代码
Nas 群晖 git
2019-01-04

用群晖Nas做站点，可以从远端拉取数据到Nas上
Mac上操作非常简单，用终端工具进到Nas需要拉取的目录git pull拉取就行了
但是如果在Windows上操作，也非常简单不过需要安装个工具
git工具到命令真的非常强大，感觉上让Windows像Linux一样用命令行，虽然那时不可能错觉也挺好
直接进到需要拉取的文件夹中，右键选择Git Bash Heregit的命令行工具就拉取git pull就行了

但是有时候这样的操作在Nas外拉取数据然后再存入Nas里面，当数据较多的时候
会出现一下严重的卡顿，不如直接在Nas里面拉取数据来得更快些

那么打开群晖Nas的ssh功能，安装git server套件给予权限，使用命令行用管理组账号登录Nas
使用命令

git config --global user.name “用户名”

git config --global user.email “邮箱”

在使用命令
ssh-keygen -t rsa
生成公匙文件
访问登录用户的公匙文件

cat /var/services/homes/用户名/.ssh/id_rsa.pub

将公匙文件放置到远程仓库的公匙列表中
这样就可以随时进入Nas拉取远程服务器的代码到Nas中完成部署站点

如果出现权限的问题
chmod 755 .ssh
改变.ssh的权限即可正常使用git clone命令

注意

只要是使用ssh都会有安全风险，要记得随用随关就好
